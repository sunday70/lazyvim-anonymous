# 💤 LazyVim

地址 [LazyVim](https://github.com/LazyVim/LazyVim).
请参阅 [文档](https://lazyvim.github.io/installation) 开始使用.

## 🛠️ 安装

#### 备份当前的 nvim 和共享文件夹

```shell
mv ~/.config/nvim{,.bak}
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}
```

#### 克隆我的 LazyVim

```shell
git clone https://github.com/LazyVim/starter ~/.config/nvim
```

#### 启动 Neovim

```shell
nvim
```

#### 注释快捷键

```shell
gcc
```

#### ⚠️ 警告

- nvim -M 模式不可用 (仅大写M)
